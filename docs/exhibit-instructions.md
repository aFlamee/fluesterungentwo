## Einschalten (Start)
1. Projektor/Beamer einschalten und korrekt ausrichten.
2. Rechner einschalten und warten bis Desktop sichtbar ist.
3. Terminal oeffnen und den Server starten: `pnpm dev` (falls bereits laeuft, ueberspringen).
4. Browser (Microsoft Edge) oeffnen.
5. `http://localhost:5173` aufrufen.
6. Mit `F11` in den Vollbildmodus wechseln und die Ausgabe auf den Beamer spiegeln/duplizieren.
7. Einmal auf **Aufnahme starten** klicken und Mikrofonzugriff erlauben.

## Nutzung
- Bitte beginne mit: "Ich kann mich daran erinnern, dass ..."
- Nach der ersten Freigabe startet die Aufnahme automatisch, sobald gesprochen wird (kein Button noetig).
- Nach ca. 1 Minute ohne Bedienung verschwindet das Bild automatisch und das System ist wieder bereit.

## Ausschalten
1. Vollbild mit `F11` beenden.
2. Browser schliessen.
3. Im Terminal den Server mit `Ctrl+C` stoppen.
4. Projektor/Beamer ausschalten.
